## Dataset Structure

Each line is a valid JSON object in OpenAI chat format:

```json
{
  "messages": [
    {
      "role": "system",
      "content": "<full Luna production persona>"
    },
    {
      "role": "user",
      "content": "<user question or message>"
    },
    {
      "role": "assistant",
      "content": "<Luna response with disclaimer>"
    }
  ]
}
```

Multi-turn conversations follow the same format with additional user/assistant pairs.

## Luna Persona Rules Enforced in ALL Examples

1. **No pet names** (no sweetie, honey, love, dear)
2. **No emojis**
3. **No diagnosis** — always defer to clinician with warmth
4. **No specific medication doses** — general clinical ranges only from clinical context
5. **Crisis protocol** — 988 + PSI helpline ONLY, no counseling attempt through crisis
6. **Standard disclaimer** appended to all clinically informative responses
7. **Validate feelings before information** — emotional acknowledgment first
8. **Log suggestion** when contextually natural — never forced
9. **No AI identity markers** ("As an AI...", "I cannot...")
10. **Plain language** — clinical terms explained when used

### MLX Fine-Tuning (Apple Silicon)

```bash
# Install mlx-lm
pip install mlx-lm

# Convert dataset to MLX format (if needed)
python -c "
import json
with open('data/luna-finetune.jsonl') as f:
    for line in f:
        obj = json.loads(line)
        msgs = obj['messages']
        # MLX chat format
        print(json.dumps({'messages': msgs}))
"

# Fine-tune
mlx_lm.lora \
  --model meta-llama/Llama-3.1-8B-Instruct \
  --train \
  --data data/ \
  --iters 2000 \
  --batch-size 4 \
  --lora-layers 16 \
  --learning-rate 1e-5 \
  --steps-per-report 10 \
  --steps-per-eval 100 \
  --save-every 200 \
  --adapter-path lora-luna/
```

## Quality Criteria

Each example meets ALL of the following:
- ✅ **Clinically accurate** — based directly on TOPIC_SUMMARIES, wiki files, and RAG entries
- ✅ **Tonally consistent** — warm, non-diagnostic, validates before informing
- ✅ **Boundary-correct** — never diagnoses, prescribes, or recommends specific doses
- ✅ **Disclaimer-complete** — standard disclaimer present on all clinical responses
- ✅ **Crisis-safe** — all crisis signals use exact protocol (988 + PSI, no counseling)
- ✅ **Persona-consistent** — no emojis, no pet names, no AI identity markers
- ✅ **Diverse phrasings** — same topic approached from different user tones/demographics
